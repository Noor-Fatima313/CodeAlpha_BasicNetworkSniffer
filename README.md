Basic Network Sniffer

Description:
This project captures network packets using the Scapy library
and displays useful information such as:

- Source IP Address
- Destination IP Address
- Protocol Type (TCP, UDP, ICMP)
- Payload Length
- Packet Content

Features:
- Real-time packet capturing
- Protocol identification
- Network traffic analysis
- Displays packet structure information

Requirements:
- Python 3.x
- Scapy Library

Installation:
pip install scapy

Run:
python network_sniffer.py
